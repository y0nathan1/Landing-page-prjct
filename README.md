# Elysian Journeys - Premium Tour and Travel Landing Page

A modern, professional landing page for a premium tour and travel company built with HTML, CSS, and JavaScript.

## Features

- **Modern Design**: Clean, professional layout with travel-themed color scheme (blues, teals, and oranges)
- **Responsive**: Fully responsive design that works on all devices
- **Interactive Elements**: Auto-sliding hero carousel, smooth scrolling navigation, mobile menu, and form handling
- **Professional Sections**:
  - Hero section with rotating background images and key statistics
  - Popular destinations showcase with detailed information
  - Featured tour packages with ratings, pricing, and booking buttons
  - About section highlighting company strengths and certifications
  - Customer testimonials with star ratings
  - Comprehensive contact form with destination selection
  - Detailed footer with organized links and social media

## Color Theme

- **Primary**: Deep Blue (#0066CC) - Trust and professionalism
- **Secondary**: Teal (#00A8A8) - Water and adventure
- **Accent**: Orange (#FF6B35) - Energy and excitement
- **Background**: Light Gray (#F8F9FA) - Clean and modern
- **Text**: Dark Blue (#1A365D) - Readability and sophistication

## Technologies Used

- HTML5 with semantic structure
- CSS3 with CSS Grid and Flexbox
- Vanilla JavaScript for interactivity
- Google Fonts (Inter) for modern typography
- Font Awesome Icons
- Unsplash for high-quality travel images

## How to View

1. Clone or download the project files
2. Open `index.html` in your web browser
3. The page is fully functional with all styling and interactivity

## Customization

- **Colors**: Modify CSS custom properties in `:root` for different color schemes
- **Content**: Update destination and tour information in `index.html`
- **Images**: Replace Unsplash URLs with your own high-resolution travel images
- **Fonts**: Change Google Fonts import for different typography
- **Tours**: Add or modify tour packages in the tours section
